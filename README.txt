# DigitalAlarmClock

一個電子時鐘應用程式，顯示台灣時間 (GMT+8)，並提供鬧鐘設定、自訂鬧鈴及操作紀錄的功能。

## 功能
- **電子顯示板**：以 12 小時制顯示目前時間，並在右下角顯示上午/下午 (AM/PM)。
- **鬧鐘設定**：讓使用者設定鬧鐘時間，並在時間到時響起鬧鈴。
- **自訂鬧鈴**：可選擇 mp3 或 wav 格式的自訂鬧鈴檔案。
- **操作紀錄**：紀錄所有操作，包含設定鬧鐘、停止鬧鈴、匯出紀錄等，並可匯出為 .txt 檔案。

## 安裝
1. 克隆此倉庫：
    ```bash
    git clone https://github.com/yourusername/DigitalAlarmClock.git
    ```
2. 在您偏好的 IDE（例如 Visual Studio）中打開專案。
3. 編譯並運行應用程式。

## 使用方法
1. **查看時間**：應用程式主畫面顯示目前時間，並即時更新。
2. **設定鬧鐘**：選擇鬧鐘時間，並設定鬧鈴檔案。
3. **停止鬧鐘**：鬧鐘響起時，點擊停止按鈕手動關閉鬧鈴。
4. **查看紀錄**：操作紀錄會顯示所有的操作歷史，可匯出為 .txt 檔案。
